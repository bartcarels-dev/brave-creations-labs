# Brave Creations Hub

A dark, modern projects hub for **Brave Creations Labs** — showcasing experimental apps and tools.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually [http://localhost:5173](http://localhost:5173)).

## Build for production

```bash
npm run build
npm run preview
```

## Projects

Edit `src/data/projects.js` to add or update projects. Set each project's `url` to link to the live app when ready.

## Stack

- [React](https://react.dev/)
- [Vite](https://vite.dev/)
